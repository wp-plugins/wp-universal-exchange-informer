Exchange rate informer for Wordpress

== Description ==

Exchange rate informer for Wordpress.
This plugin allow to get exchange rates from banks supported by plugin, and show their like widget in your blog.
Now exchange informer can be added by shortcode in text widget, page or post in your blog.

Supported banks: 
The Central Bank of the Russian Federation, 
National bank of Ukraine, 
National Bank of Moldova.

== Installation ==

1. Upload `wp-universal-exchange-informer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the "Settings | WP Universal Exchange Informer" administration page to create informer.
4. Add "WP Universal Exchange Informer" widget to sidebar and select informer needed to show in this widget or use the shortcode to install informer in text widget, page or post.
